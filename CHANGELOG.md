# 更新日志

本文件记录所有重要的项目变更，格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.2.0] - 2026-04-01

### 修复
- 修复运算符优先级错误，确保网络错误检测正确
- 修复异常处理过于宽泛的问题，避免隐藏真正的错误
- 修复 git rev-list 命令参数顺序错误，正确检测本地和远程分支差异

### 改进
- 增强路径安全验证，禁止访问系统目录
- 简化分支检测逻辑，提取 `_get_remote_branch()` 函数
- 新增 `src/constants.py` 模块，集中管理常量
- 改进日志消息，包含更多上下文信息
- 简化更新履历记录，只记录最新提交信息
- 改进网络错误检测，提供更清晰的错误提示
